주문 흐름(권장)

  1. 표준품: 상품 선택 → 주문 → 입금 확인 → 제작/출고
  2. 맞춤: 견적요청 → 사양/금액 확정 → 입금 → 제작/출고

표준 규격은 “바로 주문”

옵션이 복잡하면 구매가 멈춥니다. 온라인에는 표준 규격 10~20개만 먼저 올리고, 나머지는 견적으로 돌리는 방식이 가장 안정적입니다.

맞춤 제작은 “견적 폼”

규격/골/수량/인쇄 여부만 받아도 견적 정확도가 올라갑니다. 파일 업로드(인쇄)가 필요한 경우도 폼으로 받는 게 운영이 편합니다.

배송은 착불 고지로 단순화

부피·거리 변수(택배/퀵)를 “협의/착불”로 고지하면 온라인 운영이 현실적으로 가능합니다. 대신 출고 기준(입금확인)을 명확히 써야 합니다.
